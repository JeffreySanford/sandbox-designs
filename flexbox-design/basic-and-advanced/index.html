Instead of using the normal float left methods of the past, we can use flex box to properly format a horizonal menu. This is simply done with display: flex; targetting the class of the flex element.

Aligning the menu can be done very quickly with the property justify-content: flex-start; for left justification and justify-content: flex-end; for right alignment. Other justify-content properties include center, space-around and space-between.
